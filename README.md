# Medicraft
![Header](https://github.com/SuperBigtoo/Medicraft/blob/f1e312eda8541953f39167210add44b936ae088c/Preview%20Images/header.png)

> This repository is part of a Special Problem (Senior/Final Project).

**Medicraft (Thai Traditional Medicine Crafting RPG Game)** เป็นเกมแนว Top-down 2D RPG โดยเกมจะให้ผู้เล่นสวมบทบาทเป็นโนอาห์ซึ่งเป็นตัวละครหลัก และดำเนินกิจกรรมต่างๆตามเนื้อเรื่อง มีการต่อสู้กับมอนสเตอร์ในดันเจี้ยนเพื่อเก็บวัตถุดิบมาทำยาสมุนไพรไทย ยาแผนโบราณไทย ในการรักษาผู้ป่วย และระบบอื่นๆอย่างการสร้างไอเทม การปลูกผัก และการทำอาหาร

## Features
- **Story Mode:** ทำภารกิจเพื่อดำเนินเรื่องของเกม หรือทำการสำรวจสิ่งแวดล้อมต่างๆ ภายในเกมได้อย่างอิสระ
- **Progression:** อัพเกรดเลเวลตัวละครและเลเวล skills เพื่อเพิ่มค่าพลัง หรือดูรายการยาแผนไทยที่ปลดล็อคแล้ว
- **Creafting Item:** สร้างไอเทมต่างๆอย่าง Consumable Item, Equipment Item และยาแผนโบราณไทยในการผจญภัยและทำภารกิจ
- **Cropping:** ภายในบริเวณพื้นที่บ้านของผู้เล่นจะมีแปลงเพาะปลูกที่สามารถปลูกไอเทมพืชผลต่างๆได้
- **Drugstore Minigame:** มินิเกมเปิดร้านขายยา ทำการสร้างยาแผนไทยให้ตรงตามอาการป่วย โดยอาการป่วยของคนไข้จะถูกสุ่มจากรายการยาแผนไทยที่ผู้เล่นปลดล็อคแล้ว
